# Bookscrape-trends
Short, reproducible pipeline for book trends using the Open Library **Subjects API**. Ingest → validate → tidy → visualize.

**Link to project:** https://github.com/aliouas/bookscrape-trends

## How It's Made
**Tech used:** Python, pandas, numpy, requests, matplotlib, pytest

## TL;DR
- **Source:** Open Library Subjects API (public, no key)
- **Scope:** ~[N] works across [K] subjects (e.g., science_fiction, fantasy)
- **Output:** tidy CSV + trend figures
